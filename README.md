1. 짜장면 레시피 (표준 예시)
핵심 포인트: 춘장의 고소함과 양파의 단맛을 극대화할 것.
재료: 춘장 100g, 돼지고기(등심) 50g, 양파 1개, 양배추 1/4통, 설탕 1큰술, 전분물 약간.
조리 순서:
팬에 기름을 충분히 두르고 춘장을 튀기듯 볶아 따로 덜어둔다.
돼지고기를 먼저 볶아 기름을 내고, 다진 생강과 간장으로 향을 입힌다.
양파와 양배추를 넣고 센 불에서 빠르게 볶아 수분을 잡는다.
볶아둔 춘장과 설탕을 넣고 잘 섞은 뒤, 전분물로 농도를 조절한다.
2. 짬뽕 레시피 (표준 예시)
핵심 포인트: 강한 불맛(화근)과 시원하고 칼칼한 국물.
재료: 돼지고기 30g, 오징어 1/2마리, 새우 3마리, 고춧가루 2큰술, 채소(배추, 양파, 목이버섯), 사골 육수.
조리 순서:
팬에 기름을 두르고 돼지고기와 파, 마늘을 볶아 향을 낸다.
고춧가루를 넣고 타지 않게 볶아 고추기름을 만든다. (불맛의 핵심!)
해산물과 채소를 넣고 센 불에서 빠르게 볶아 불향을 입힌다.
육수를 붓고 끓어오르면 소금과 후추로 간을 맞추어 마무리한다.

2. 짜장면 레시피
2-1. 재료
재료	분량 예시
춘장	100g
돼지고기	80g
양파	1개
양배추	1/4통
대파	1대
다진 마늘	1큰술
식용유	적당량
설탕	1큰술
간장	1큰술
전분물	2~3큰술
면	1인분
2-2. 단계별 조리
1.	춘장 볶기
기름에 춘장을 약불~중불로 볶아 쓴맛을 줄이고 고소한 향을 살립니다.
2.	고기 볶기
돼지고기를 먼저 볶아 기름과 감칠맛을 냅니다.
3.	채소 볶기
양파, 양배추, 대파, 마늘을 넣고 센 불에서 빠르게 볶아 수분을 잡습니다.
4.	양념 넣기
볶은 춘장, 간장, 설탕을 넣고 잘 섞어 맛의 균형을 맞춥니다.
5.	농도 조절
전분물을 넣어 소스의 점도를 맞춘 뒤 한 번 더 끓입니다.
2-3. 완성 팁
•	짜장면은 채소의 단맛이 중요하므로 양파를 충분히 볶아야 합니다.
•	춘장은 반드시 한 번 볶아 사용해야 깊은 맛이 납니다.
•	소스가 너무 묽으면 면에 잘 안 감기므로 농도 체크가 중요합니다.

3-1. 재료
재료	분량 예시

육수	600ml
소금	약간
후추	약간
면	1인분
3-2. 단계별 조리
1.	향신기름 만들기
팬에 기름을 두르고 대파, 마늘, 돼지고기를 볶아 향을 냅니다.
2.	고춧가루 볶기
고춧가루를 넣고 타지 않게 빠르게 볶아 고추기름과 불맛을 만듭니다.
3.	재료 볶기
오징어, 새우, 양파, 양배추를 넣고 센 불에서 빠르게 볶습니다.
4.	육수 넣기
준비한 육수를 붓고 끓여 국물 맛을 완성합니다.
5.	간 맞추기
소금, 후추로 마무리 간을 맞추고 한 번 더 끓입니다.
3-3. 완성 팁
•	짬뽕은 불맛이 생명입니다. 재료를 오래 익히지 말고 빠르게 볶아야 합니다.
•	고춧가루는 타지 않게 볶아야 쓴맛이 나지 않습니다.
•	해산물은 오래 끓이면 질겨지므로 마지막에 넣는 순서를 지켜야 합니다.

4. 짜장면 vs 짬뽕 비교표
항목	짜장면	짬뽕

•	짬뽕은 불맛과 국물의 균형이 핵심이다.
•	둘 다 센 불 조리와 빠른 동작이 중요하다.
•	맛은 재료보다 순서와 불 조절에서 완성된다.
